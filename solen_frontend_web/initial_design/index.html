Solen design system

  • Palette: Warm amber on off-white paper. One accent — the sun gold (oklch 68% 0.16 75). Muted warm neutrals for body.
  • Typography: Iowan Old Style / Georgia for serif display + system sans for body. Editorial posture, generous leading.
  • Layout: Asymmetric grids, generous whitespace, borders do the work (no shadows, minimal rounding).
  • Navigation: Animated spring-dropdown with warm avatar, section labels, nav icons.
  • Mood icons: 5 custom sun-face SVGs — stormy to radiant, using the amber palette.
  • UX flows: Category tree browser in onboarding; mood → reflection check-in flow; feed + card views on Inspire.
